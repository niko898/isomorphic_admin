# File Manager

File and document management system.

## Routes

```typescript
routes.file = {
  dashboard: '/file',
  manager: '/file-manager',
  upload: '/file-manager/upload',
  create: '/file-manager/create',
};
```

## Features

- File upload/download
- Folder organization
- File preview
- Storage analytics
- Team member management
- Recent files
- Activity tracking

## Integration

Uses Uploadthing for file uploads.
