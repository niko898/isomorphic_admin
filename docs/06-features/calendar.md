# Event Calendar

Event and appointment management calendar.

## Routes

```typescript
routes.eventCalendar = '/event-calendar';
```

## Features

- Event creation
- Event scheduling
- Calendar views (month, week, day)
- Event editing
- Recurring events

## Libraries

- **react-big-calendar 1.8.5**
- **date-arithmetic 4.1.0**
- **dayjs 1.11.10**

## Hook

```typescript
import { useEventCalendar } from '@/hooks/use-event-calendar';
```
